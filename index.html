What is Bootstrap?

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Is it necessary to use Bootstrap in your web projects?

Yes, because bootstrap eliminates the use of libraries that always differ from one developer to the other. Thus, you get to maintain the stability of your project elements no matter who is tackling them.
